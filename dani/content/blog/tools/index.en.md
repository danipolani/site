As I've been working on my language, [Lemu Teloku]({{< ref "/worldbuilding/lemuteloku" >}}), as well as other projects, I've created several tools for my own use. Then I've converted some of them into publicly available services. Here is a list of them. Feel free to use.

This page is **evergreen** - I’ll add entries as new projects ship.

## Language, linguistics and conlanging

### Generative Grammar Engine

A visual editor for **generative grammars**: rules, parse trees, export. Handy for playing with formal grammars and seeing how generative rules behave. Inspired by Tracery and mostly compatible with it's grammars, though Tracery code wasn't used in the project.

In addition to grammar expansion functionality, it supports different randomization strategies, saves generation path for each expansion, allows to freeze parameters to get more precise results.

- [Project page]({{< ref "/projects/grammar" >}})
- [grammar.tinygods.dev](https://grammar.tinygods.dev/)

### Bitext Align

A **word-by-word translation visualizer** ([Bitext Align](https://aligner.tinygods.dev/)): link words on the source and target sides, optionally add an **interlinear gloss** row, then export (PNG, SVG, PDF, HTML). Useful for lessons, articles, and social posts when you need a clear **bilingual sentence alignment** graphic.

- [Project page]({{< ref "/projects/aligner" >}})
- [aligner.tinygods.dev](https://aligner.tinygods.dev/)
