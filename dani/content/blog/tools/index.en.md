A running list of tools I build or maintain. This page is **evergreen**—I’ll add entries as new projects ship.

## Generative Grammar Engine

A visual editor for **generative grammars**: rules, parse trees, export. Handy for playing with formal grammars and seeing how generative rules behave.

- [Project page]({{< ref "/projects/grammar" >}})
- [grammar.tinygods.dev](https://grammar.tinygods.dev/)

## Bitext Align

A **word-by-word translation visualizer** ([Bitext Align](https://aligner.tinygods.dev/)): link words on the source and target sides, optionally add an **interlinear gloss** row, then export (PNG, SVG, PDF, HTML). Useful for lessons, articles, and social posts when you need a clear **bilingual sentence alignment** graphic.

- [Project page]({{< ref "/projects/aligner" >}})
- [aligner.tinygods.dev](https://aligner.tinygods.dev/)
